## Run Locally

Clone the project

```bash
  git clone https://github.com/dvirbo/Ex2_OOP.git
```

Go to the project directory

```bash
  cd Ex2_OOP
```

Start the server

```bash
  java -jar Ex2.jar G1.json 
```
